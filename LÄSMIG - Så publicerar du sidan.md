# Så publicerar du din hemsida på GitHub Pages (GRATIS)

Din sida kommer ligga på: **https://naweed-rahimi.github.io**

---

## Steg 1 — Skapa ett gratis GitHub-konto

1. Gå till https://github.com
2. Klicka på "Sign up"
3. Välj användarnamn: **naweed-rahimi** (viktigt – detta blir din URL!)
4. Ange din e-post och ett lösenord
5. Verifiera ditt konto

---

## Steg 2 — Skapa ett nytt repository

1. Logga in på GitHub
2. Klicka på den gröna knappen **"New"** (eller gå till github.com/new)
3. Döp repository till exakt: **naweed-rahimi.github.io**
   *(det måste heta exakt så för att GitHub Pages ska fungera)*
4. Välj **"Public"**
5. Klicka **"Create repository"**

---

## Steg 3 — Ladda upp din fil

1. På din nya repository-sida, klicka **"uploading an existing file"**
2. Dra och släpp filen **index.html** från mappen på skrivbordet
3. Scrolla ner och klicka **"Commit changes"**

---

## Steg 4 — Aktivera GitHub Pages

1. Gå till **Settings** (kugghjulet) i ditt repository
2. Scrolla ner till **"Pages"** i vänstermenyn
3. Under "Branch", välj **main** och klicka **Save**
4. Vänta 1–2 minuter

---

## Steg 5 — Din sida är live! 🎉

Gå till: **https://naweed-rahimi.github.io**

Sidan dyker upp i Google och Safari inom några dagar.

---

## Uppdatera sidan i framtiden

Om du vill ändra något på sidan:
1. Redigera filen index.html
2. Ladda upp den nya versionen till GitHub (samma steg som ovan)
3. Sidan uppdateras automatiskt inom 1–2 minuter
